# quick sort
## my goal with this project
to learn how quicksort works and implement in c++
## what i learned
quicksort works with a pivot element, partitioning the array so everything smaller is on the left side and larger on the right, then recursively sorts both sides
quicksort sorts in place, the list you have is directly manipulated
## challenges
indexing my loops for subarrays in recursion
messed up final swap and swapped with pivot_value instead of v[i+1]
## what i would do differently
figure out my indexing by drawing out the sort and labelling everything.
